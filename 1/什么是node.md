# 怎么打开node
- 需要在终端运行  => 怎么打开终端 
- - 1、通过window Powershell 打开终端
- - 2、通过window + r 打开运行界面  输入指令 cmd 打开终端
- - 3、通过文件系统的路径  输入cmd 打开终端
- - 4、通过vscode打开终端

# 常见的终端指令
- cd 改变目录
- mkdir||md 创建目录
- del | rm 删除目录
